ZentroTask

Real Time Task Manager Web Application

Hover Over the image to see image description

ZentroTask is a real-time task management application built to simulate enterprise-level team coordination and data-driven decision making. It showcases seamless user and task management through live updates and dynamic visualizations.


Key Functionalities:

  • Real-time CRUD operations for both tasks and users using Socket.io.
  • Interactive dashboard with live charts powered by Recharts.
  • Fully documented API endpoints using Swagger for streamlined development and testing.
  • Deployed on Render for accessible live preview and hosting.

Home Page Highlights:

  • Pie Chart: Shows ratio of completed vs. pending tasks for an instant project overview.
  • Bar Chart: Displays how tasks are distributed among users, clarifying workload balance.
  • Live Sync: All visualizations update instantly as CRUD actions occur, without page reloads.

Users Page:

  • View all users in a responsive table layout.
  • Add new users via modal forms using React Hook Form with Yup Resolver for strong validation.
  • Edit user details with changes reflected across all clients instantly.
  • Delete users with a confirmation prompt to prevent accidental data loss.

Tasks Page:

  • Create tasks with title, description, status, and user assignment.
  • Edit and mark task completion in real-time.
  • Delete outdated tasks with confirmation.
  • Real-time updates broadcasted to all connected users.
